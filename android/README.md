# ARC eye VLSDK for Android

## Overview

ARC eye VLSDK for Android is a native Android library that estimates the position of mobile devices in space using the ARC eye Visual Localization (VL) API. This enables easy implementation of AR apps that utilize spatial information.

## Quick Start

1. Open `Android/VLSDKArcEyeApp` in Android Studio.
2. In the source, locate the `VLSDKService` initialization and replace the placeholders with your issued credentials:
   - `location` — your location name
   - `invokeUrl` — your VL API URL
   - `secretKey` — your VL API key
3. Build and run on a device.
