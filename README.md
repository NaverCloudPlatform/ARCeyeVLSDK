# ARC eye VLSDK

ARC eye VLSDK estimates the 6-DoF pose of a mobile device from a single camera image, using the ARC eye Visual Localization (VL) API. Use it for AR apps that need accurate device pose where GPS is unreliable, such as indoor navigation.

## Repository structure

| Directory               | Platform                       |
| ----------------------- | ------------------------------ |
| [`android/`](./android) | Native Android (Kotlin / Java) |
| [`ios/`](./ios)         | Native iOS (Swift)             |
| [`unity/`](./unity)     | Unity (iOS / Android)          |

See the README inside each directory for installation and usage.
